## Visão geral

O Model Context Protocol (MCP) é um padrão aberto que permite que aplicações de IA conversem com serviços externos — como nossa documentação Mintlify e as APIs públicas da Husky. A partir do conteúdo deste repositório, a Husky publica um servidor MCP que libera ferramentas de busca na documentação e acesso seguro a endpoints selecionados das nossas APIs:

- **API Pública Husky**
- **API Open Delivery**

Este guia explica como funciona o servidor MCP da Husky, como habilitar novos endpoints e como seus usuários podem se conectar a partir dos principais clientes MCP (Claude, Cursor, VS Code e outros).

## Onde encontrar o MCP da Husky

Você pode conectar nosso MCP direto a sua IDE, ou também pode usar através do **Context7**

### Usando MCP da Husky

Para conectar nosso MCP basta procurar ao lado do titulo da página e clicar para conectar a sua IDE.
